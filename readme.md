### 欢迎大家访问 CopWS 的介绍界面！

CopWS 是一个资源站，作者copcin，小学生一枚~

CopWS 的灵感起源于怀揣了很久的一个想法，我因为酷爱折腾，所以系统镜像这种东西是必须要有的~ 但由于我折腾的系统大部分是Linux，MSDN的镜像又大部分是Windows，所以，我来了一个灵感，那就是把所有常用的系统镜像整理成一个资源站，供自己和他人使用。但是当时特别悲哀的是，我连HTML是什么都不知道，不会前端，写网站这种事自然就不存在了~

终于，我下定了决心，用2天时间“啃”完了HTML+CSS，这才有了 CopWS。由于只是爱好者，ui 写的难看一点请见谅，资源也会尽快扩充~

关于 CopWS 的详细信息，请访问 https://copws.github.io/about.html

希望你们在 CopWS 玩的开心！
