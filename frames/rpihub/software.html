教程频道
  • SD Card Formatter
    （TF 卡格式化工具，由于Windows自带的格式化功能不是专门针对 TF 卡的，所以需要一个额外的 TF 卡格式化工具）
  • Raspberry Pi Imager
    （官方提供的TF卡烧录工具，适配树莓派，新手推荐）
  • balenaEtcher
    （一个功能强大，界面简洁的TF卡烧录工具，不同的是，它也可烧录系统到U盘）
  • 树莓派 Windows 驱动
  • 树莓派 UEFI 固件
    （把树莓派 UEFI 固件放进TF卡的EFI分区，即可让树莓派得到UEFI引导）
  • NOOBS
    （全称 New Out Of Box Software，是树莓派一个全新的安装系统，允许用户在树莓派上安装一个或多个不同的Linux或Windows loT系统）
  • WoR 部署工具
    （全称 Windows on Raspberry ，把 Windows on ARM 写进TF卡的一件必备的工具）
