关于 CopWS 1.0.0 beta (RC)

CopWS 是一个资源站，作者copcin，小学生一枚~

CopWS 的灵感起源于怀揣了很久的一个想法，我因为酷爱折腾，所以系统镜像这种东西是必须要有的~ 但由于我折腾的系统大部分是Linux，MSDN的镜像又大部分是Windows，所以，我来了一个灵感，那就是把所有常用的系统镜像整理成一个资源站，供自己和他人使用。但是当时特别悲哀的是，我连HTML是什么都不知道，不会前端，写网站这种事自然就不存在了~

于是，我开始看狂神的前端视频，仅仅用了一天，就“啃完”了 HTML 的所有课程。CopWS　beta 1 也因此而生。但是由于 beta 1 是用纯 HTML 5 写的，UI 这种东西，根本就没有（T_T）于是，我又花了一天时间，啃完了 CSS ，CopWS beta 2 因此改进了 UI。由于我喜欢折腾树莓派，CopWS beta 4 中，我又新增了 RPi Hub 模块，里面收录了很多树莓派 镜像&工具~ 后续的 beta 5 & 5.1 又进行了改进和扩充~

可是beta 5的UI相对于其它网站，感觉还是逊色太多了。这个原因让我有了再次改进 UI 的想法。一个前端初学者，改一个大网站（至少在我心里它很大）并非易事。在改进的过程中，我尝试了三次，前两次都以失败告终。在这第三次里，我重新梳理了 CopWS 的整个结构，且参考了很多我心目中的“偶像网站”，如ibeta尝鲜派，next.itellyou.cn等。功夫不负有心人，在第三次尝试中，我终于把 CopWS 做成了一个网站的样子，虽然不是非常漂亮，但它起码从一组 HTML 文件变成了一个小站。这就是 CopWS beta 6。

在 CopWS beta 7 里，我也如出一辙的把 RPi Hub 的 UI 改完了。目前 RPi Hub 已经进入 RC （预发布阶段）了，主部分也在 beta 7.1 之后的一个版本进入 RC 了。 1.0 指日可待~

在将来的 CopWS 里，我还会继续扩充它的资源，改进它的 UI，让 CopWS 成为一个像 MSDN 一样成熟的资源站。最后再强调一次，现在 CopWS 还处于测试阶段，且持续更新。不过 1.0 正式版本应该快了，敬请期待！

最后，感谢支持 CopWS 的你们。我会为搭建一个更好的 CopWS 而继续努力的~
