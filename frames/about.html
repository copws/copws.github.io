关于 CopWS 1.0.0 (Stable)

CopWS 是一个资源站，作者copcin，小学生一枚~

CopWS 的灵感起源于怀揣了很久的一个想法，我因为酷爱折腾，所以系统镜像这种东西是必须要有的~ 但由于我折腾的系统大部分是Linux，MSDN的镜像又大部分是Windows，所以，我来了一个灵感，那就是把所有常用的系统镜像整理成一个资源站，供自己和他人使用。但是当时特别悲哀的是，我连HTML是什么都不知道，不会前端，写网站这种事自然就不存在了~

于是，我开始疯狂学习前端知识，在 4 天里学完了前端三件套，并发布了第一个 CopWS 版本 0.1 (beta 1)。接着我又用 CSS 对 CopWS 进行了一个基础的 UI 优化。在 beta 4 里，我又给 CopWS 增加了 RPi Hub 模块，给喜欢折腾树莓派的人提供一个资源收集。这样一直持续到了 beta 5.1~

接下来就是 CopWS 的两次脱胎换骨：beta 6 和 beta 7。 由于 beta 5 & 5.1 感觉就是加了 CSS 的 HTML 文档，所以我不再更新一次 UI 是不行的。beta 6 中，我把整个网站重新布局，并加入了一些比较复杂的组件（虽然在 web 老码农看起来只是小 case）最终，CopWS　终于从加了　CSS 的 HTML 变成了一个网站~ beta 7 也和 beta 6　一样，对 RPi Hub 进行了 UI 更新~

又经过了两个 beta 版本（一个 7.1，一个 RC），一共 11 个 beta 版本，CopWS　正式版终于和大家见面了。这次更新正式版预计需要很久才有下一个　beta 测试版本，要不为什么叫正式版呢~

在将来的 CopWS 里，我还会继续扩充它的资源，改进它的 UI，让 CopWS 成为一个像 MSDN 一样成熟的资源站。另外提前剧透一下 2.0 的内容，可能会采用仿尝鲜派 betahub.cn 的 UI~

最后，感谢支持 CopWS 的你们。我会为搭建一个更好的 CopWS 而继续努力的~
